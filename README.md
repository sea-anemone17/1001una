# 1001una v2 - 1차 기능 완성본

정적 GitHub Pages 배포를 전제로 만든 영어 문법/어법 학습 앱입니다.

## 포함 기능

- 홈 / 챕터 / 문장 분석
- 문법 지도 + 세부 태그 연결
- 적용 훈련 + 정답/오답 피드백
- 메타인지 진단 기록
- 대시보드
- 복습 추천
- 챕터별 심화 관문 샘플
- 별 기록 / 자기평가 보상
- 로컬 저장
- 백업 내보내기 / 가져오기 / 초기화

## 실행

정적 서버에서 실행하세요.

```bash
python -m http.server
```

또는 VS Code Live Server를 사용할 수 있습니다.

## 사용자 기록

사용자 기록은 `localStorage`의 `1001una_v2_user_data`에 저장됩니다.
콘텐츠 데이터와 분리되어 있으므로 앱 파일을 업데이트해도 기록이 바로 덮어씌워지지 않습니다.
